# GreenFood Challenge Project
For our CMPT 276 class we were tasked with creating an app that can help regular people lower their Co2 emmisions by changing their diet!

### How to use 
Currently the app is not avaliable on the Android app store. The only way to use it is by forking the repository and porting the app to your Android device.
